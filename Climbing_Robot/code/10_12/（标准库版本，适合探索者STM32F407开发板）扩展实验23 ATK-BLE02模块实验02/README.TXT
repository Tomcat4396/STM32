实验器材:
探索者STM32F407开发板

实验目的:
学习ATK- BLE02蓝牙串口模块的使用。

硬件资源:
1, DS0、DS1(连接在PF9、PF10)
2, 串口1(波特率: 115200, PA9 / PA10连接在板载CH340上)
3, 按键KEY0(PE4) / KEY1(PE3) / KEY2(PE2) / KEY_UP(PA0, 也称之为WK_UP)
4, ATK - BLE02模块2个.
5, USB串口模块1个

ATK-BLE02      探索者STM32F407开发板
VCC < ------------ > 5V
GND < ------------ > GND
TXD < ------------ > PB11
RXD < ------------ > PB10
STA < ------------ > PF6
WKUP < ----------- > PC0

模块透传（主设备），实现主设备与从设备连接实现透传。
实验使用两个模块。从设备模块用USB串口模块连到电脑，并需用上位机读取模块的MAC地址。
主设备通过扫描会搜索到周围从设备MAC的地址，并输出到串口。主设备连接该从设备MAC地址，
连接成功后可通过串口助手实现数据传输。从设备发送“+LEDD0:ON”字符串，主设备板上LED0点亮，
发送“+LEDD0:OFF”字符串，熄灭LED0。按下开发板KEY1按键，可发送数据给从设备。
